## Overview
A next‑generation audio–visual simulation engine fusing gravity physics, harmonic synthesis, and interactive HUD aesthetics. Designed for mobile and desktop browsers with optimal performance and visual fidelity.

## Core Features
- Real‑time particle system (up to 5 million entities)
- Audio playback linked to particle birth and decay
- Thematic UI presets (21 skins)
- Modular PWA architecture with offline support
- Secure push‑to‑start audio gateway for mobile
